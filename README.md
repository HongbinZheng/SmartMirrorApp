# SmartMirrorApp

This is a repo for smart-mirro phone app, please follow https://docs.expo.io/versions/latest/sdk/google-sign-in/ to setup credential first.
<br/><br/> In order to run this app, you need to download expo-cli to run it. If you want to run it in expo-client, please go to branch expoClient
<br/> Otherwise, this required build standalone app for full google sign in function

<br/><br/> After clone the app, run`npm install`, then use `expo start` to start at expo client or `expo build:android` to build android package and `expo build:ios` to build ios app(this required ios developer account)

Frontend of the Smart-Mirror(Mirror-version) will be at repo https://github.com/itjinshan/Smart-Mirror

Nodejs backend(handle google api call) will be at repo https://github.com/HongbinZheng/SmartMirrorBackendNode
